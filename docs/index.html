design-lookbook

AI가 참조하는 번호 기반 디자인 컨셉 라이브러리. 각 프리셋은 토큰 · 컴포넌트 규칙 · 레이아웃 레시피 · 모션 · 보이스&톤까지 담은 풀 세트이며, 아래 카드에서 컨셉당 5화면 데모를 바로 열 수 있다. 3번으로 만들어줘가 작동하는 이유.

GitHub 저장소의 docs/ 폴더가 GitHub Pages로 배포되는 이 갤러리다. 스펙 원본(presets/<번호>-<코드네임>/SPEC.md·tokens.css)이 source of truth이며, 갤러리는 그 레퍼런스 구현이다.
