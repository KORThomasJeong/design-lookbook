starship

궤도 위성 관제 시스템의 코어 런타임입니다. 텔레메트리 수집, 명령 스케줄링, 지상국 동기화를 하나의 프로세스에서 처리합니다.

설치

npm install 후 npm run dev로 로컬 관제 콘솔을 실행합니다. Node 20 이상이 필요합니다.

주요 기능

  • 초당 2,400건 텔레메트리 스트림 처리
  • 지상국 고흥·제주 이중화 핸드오버
  • 명령 큐 재전송 및 서명 검증
